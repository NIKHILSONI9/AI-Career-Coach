# 📌 Overview

AI Career Coach is a modern AI-powered career guidance platform designed to help students and professionals make smarter career decisions through:

- 🤖 AI-based Career Recommendations
- 📄 ATS Resume Analysis
- 🧠 Skill Gap Detection
- 💬 AI Mentor Chatbot
- 📝 Cover Letter Generation
- 📊 Industry Insights Dashboard
- 🛠 Resume Builder
- 🎯 Personalized Career Roadmaps

The platform leverages **Google Gemini AI**, **Next.js**, **MongoDB**, and modern web technologies to deliver intelligent and personalized career guidance.
